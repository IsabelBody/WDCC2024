## How to run 
Requirements: 
- Python 
- **pip install dijkstar** in terminal to run the dijkstra_implementation.py file
- NodeJS

Start Two Terminals

Terminal 1
`cd frontend` 
`npm install`
`npm run dev`

Terminal 2
`cd backend`
`python app.py`
